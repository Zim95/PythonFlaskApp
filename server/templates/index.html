Data Dashboard

About this page

This page serves as a data visualization tool for various data obtained in the form of CSV files. The data obtained may be time-series data or general test data used for analytics and machine learning purposes. The webpage is a single page application that switches content upon clickable links by using jquery. It is done so, to reduce the time to design an entirely new page and also reduce the time of having to load an entire new page upon clicking the links.

About Backend

The backend serves a restful api which takes in data from a csv file. The restful web service only takes GET requests so any other request method will result in an error. The restful web service provided by the backend is an extremely basic API built upon FLASK and PYTHON 3 that does not provide JWT Authentication or any other HTTP method other than GET. The routes to the API may be added as per will. The CSV parser library created will automatically convert it to JSON and serve it. There is no support for CORS either, which means for now, the frontend and backend must run on the same machine or atleast the same origin. To run backend navigate to server folder and run server.py. Detailed instructions are provided in README.md

About Frontend

The front-end is built upon bootstrap 3 and jquery. The front-end makes a repeated call to the backend api on a particular interval. Therefore, any updates on the csv file on backend will result in the change in data on the front-end. The charts are created using CHART.JS. To run front-end, make sure that it is hosted. AJAX requests cannot be made on file:/// protocol and requires HTTP protocol. Run a local apache server and then host it there. Detailed explanation is provided in README.md.

Instructions

To view any of the data dashboards, go to "Dashboard" on the side bar. It is a dropdown menu and upon clicking will result in a lisst of available dashboards. Currently there is only a single dashboard but more dashboards can be added as per will. Clicking on Main page will take you to this page.
